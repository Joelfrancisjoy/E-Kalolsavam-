## ğŸ”„ Re-seeding the Database

To clear and re-seed the database with fresh data:

```bash

python manage.py seed_database --clear
```

This will:
- Clear all existing data (except superuser)
- Create fresh users, schools, and events
- Generate new registrations and scores
- Publish results for completed events
